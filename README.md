# dart-web-api-mysql-simple

## Description
Serves a basic api.

## Tech stack
- dart
  - http

## Docker stack
- google/dart
- mariadb

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
